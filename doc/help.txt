*Pine Editor Help Doc*

Key mapping

Ctrl+X Ctrl+X  Exit or cancel
Ctrl-/  Help

Navigation
Ctrl-/  Next page       Ctrl-B  Next buffer
Ctrl-Z  Prev page       
Ctrl-R  Open file       Ctrl-A  Go to beginning of current line
Ctrl-O  Save file       Ctrl-E  Go to end of current line

Edit
Ctrl-K  Delete current line

Command Ctrl-X
Ctrl-X k  Kill current buffer
